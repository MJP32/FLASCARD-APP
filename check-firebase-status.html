üö® The Core Issue

If allow read, write: if true; doesn't work, it means:

  1. The rules aren't being applied to your database
  2. OR the database itself has issues
  3. OR you're somehow looking at the wrong project/database

This is NOT a normal Firestore behavior. Even the most restrictive projects should work with completely open rules.
